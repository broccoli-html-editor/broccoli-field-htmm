# broccoli-field-htmm

_broccoli-field-htmm_ は、 _broccoli-html-editor_ に カスタムフィールド "htmmフィールド" を追加する拡張パッケージです。

## インストール - Install

```
$ npm install broccoli-html-editor --save
$ npm install broccoli-field-htmm --save
```
